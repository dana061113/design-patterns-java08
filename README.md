# Design Patterns Java

## 📌 Patterns used:
- Decorator
- Adapter

---

## ☕ Decorator
Dynamic adding of ingredients to beverages:
- Milk
- Sugar
- Whipped Cream

Supports:
- Espresso
- Tea
- Latte
- Mocha

---

## 💳 Adapter
Integration of different payment systems:
- PayPal
- Stripe (via Adapter)
- Crypto (via Adapter)

---

## ▶️ How to run
1. Open project in IntelliJ IDEA
2. Run:
   - decorator.Main
   - adapter.Main
